# 随码 · 品牌与传播

## 产品名

| 用途 | 名称 |
|------|------|
| 中文产品名 | **随码** |
| 英文代号 / 工程名 | AnytimeVibe（仓库、包名、协议标识保留） |
| 桌面客户端 | **随码**（安装后程序名） |
| 一句话定义 | 随时续上本机 Codex 任务的远程安全工作台 |

**命名含义**：人可以离开工位，代码任务可以「随时接上」——随身、随时、随手续码。

## 主 Slogan（推荐置顶）

### 首选主 slogan

> **离开电脑，任务不用停。**

最适合登录页大标题、应用商店简介、官网首屏。场景清晰，记忆成本低。

### 副 slogan / 品牌锁语

> **随时续上你的代码。**

适合顶栏、托盘窗口、PWA 副标题、分享卡片描述。

### 安全向副句

> **云端只传密文，代码留在本机。**

适合安全说明、技术博客、企业试用介绍。

## 品牌视觉要点

- 主色：暖纸色 `#f2eadb` + 墨色 `#17211b` + 锈橙 `#e25832` + 苔绿 `#2d7653`
- 图标：终端窗口 + 橙色提示符 + 绿色光标（「在线续写」意象）
- 语气：利落、可信、偏开发者口语，少用空泛「赋能」「一站式」
