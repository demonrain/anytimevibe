# 随码 35 秒宣传片 V2 脚本

## 0–5 秒

灵感不等你回到工位。代码，也不必。

## 5–10 秒

地铁、咖啡店、机场或家里，打开手机，选择 Codex、Claude Code 或 Grok Build，随手发起任务。

## 10–15 秒

任务在你自己的 Windows 或 macOS 电脑上执行。源码、凭据和工作区仍留在本机。

## 15–20 秒

刷新、退出、换浏览器，任务仍在处理中；新的命令进入持久队列，完成后主动通知。

## 20–25 秒

回到电脑，一键接力到原生 CLI。同一引擎、同一会话、同一权限，无需重新解释上下文。

## 25–30 秒

真实 Diff、阶段日志和完成结果继续同步回手机，下一步随时接着走。

## 30–35 秒

随码。随时随地 Vibe Coding。移动端与电脑端，无缝衔接。
