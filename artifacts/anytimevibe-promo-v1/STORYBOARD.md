# 随码宣传片分镜

| 时间 | 画面 | 运动 | 屏幕文字 |
|---|---|---|---|
| 0–5s | 深墨背景、巨大品牌宣言、橙色终端提示符 | 标题分层推入，状态光点呼吸 | 离开电脑，任务不用停 |
| 5–10s | 手机输入具体任务并点击发送，左侧远程电脑 Codex 收到相同命令 | 命令气泡进入，信号点从手机移动到电脑，终端进入执行状态 | 手机下发命令，电脑立即执行 |
| 10–15s | 延续上一幕的手机任务，左侧任务卡、右侧电脑 CLI，中间接力箭头 | CLI 紧接手机下发场景恢复同一 thread 和权限，结果箭头返回手机 | 一键接力，来回无缝 |
| 15–20s | 任务队列、处理中状态、完成标签 | 列表错峰进入，队列逐项推进 | 队列清晰，状态不丢 |
| 20–25s | Full Access 权限选择、密钥留在本机、安全边界 | 权限控件放大锁定，边界线绘制 | 权限由你控制 |
| 25–30s | 图标、产品名、CTA | 标志缩放落位，标语逐字显现 | 随码 · 随时续上你的代码 |

## Asset Audit

- 品牌图标：`assets/icon.svg`
- 产品界面：使用 HTML/CSS 重建关键状态，不依赖外部截图
- 字体：系统中文字体 + Rockwell + Cascadia Code
- 音频：原创 108 BPM 产品介绍风格 BGM，由 `scripts/generate-bgm.mjs` 确定性生成；场景切换点带轻量提示音，无旁白
